# taxi-meter

## 出租车计价：

不大于2公里时只收起步价6元。
超过2公里的部分每公里收取0.8元。
超过8公里的部分，每公里加收50%长途费。
停车等待时加收每分钟0.25元。
最后计价的时候司机会四舍五入只收到元。
